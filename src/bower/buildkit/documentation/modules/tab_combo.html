Tab Combo

The Tab Combo module is very similar to our Drawer module—it helps you organize and navigate tab-like content in a single container. This module is made of three other modules:

  1. Accordion: used to expand and collapse content that is broken into logical sections, like tabs.
  2. Tab Slider: a tab module that has the visual effect of a slider
  3. Product Highlight: a product display with an image and description

Just like the drawer module, this can have multiple columns:

  • 2 columns: tab-combo--two
  • 3 columns: tab-combo--three
  • 4 columns: tab-combo--four
  • 5 columns: tab-combo--five

You can link directly to a specific section of the tab combo with the following query string params:

  • tab: the ID of the active tab
  • accordion: the ID of the active accordion
  • tab-slider: the ID of the active tab slider

Example query string: ?tab=tab-imagewrap&accordion=accordion-panel-paper&tab-slider=tab-slider-premium-lustre.

Example query string for more than 1 option selected: ?tab=tab-softcover&tab-slider=tab-slider-premium-lustre,tab-slider-standard-landscape,tab-slider-hardcover--dust-jacket.
