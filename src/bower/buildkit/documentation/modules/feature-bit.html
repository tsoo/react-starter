Feature Bit

Feature bitting (or toggling) is a way of delivering new features to users safely, allowing for continuous delivery without relying on cherry-picking to get new features out.

For a new feature that isn't ready to go out and exposed to 100% of our users, you can "feature bit" it. Create the feature bit in Support Tools by giving it a unique name (please prefix with web-).

To show a module or group of modules if a feature bit is "enabled", wrap the module(s) in a div: 

<div class="hidden" data-feature-bit="web-test-feature" data-options="show-if-enabled:true">
  <!-- modules go here -->
</div>

To hide a module or group of modules if a feature bit is "enabled", use this code: 

<div data-feature-bit="web-test-feature" data-options="show-if-enabled:false">
  <!-- modules go here -->
</div>

For performance, we store the data from the API in the browser's sessionStorage. If you want to disable that (good to do when you turn something onto 100%), you can pass in an extra option: data-options="show-if-enabled:true;ignore-session-storage:true". By default, ignore-session-storage is false. This is optional and rarely needs to be included.

In summary, data-options has two options:

  1. show-if-enabled: true or false (optional). Defaults to true.
  2. ignore-session-storage: true or false (optional). Defaults to false.

Examples

To add new content that isn't ready to be shown to users, keep the feature bit off, using the hidden helper class to hide the element (prevents the jarring effect of it appearing on initial page load and abruptly disappearing). By using data-options="show-if-enabled:true", you're able to show the content as soon as the feature bit is turned on. 

<div class="hidden" data-feature-bit="web-holiday-promo-2016" data-options="show-if-enabled:true">
  New content
</div>

The advantage of using a feature bit here is the content can be shown when ready, without having to redeploy the application. For instance, you might have copy related to a promotion, but you don't want it visible until the promotion is active.

To hide old content while showing new, experimental content:

<div data-feature-bit="web-copy-2016" data-options="show-if-enabled:false">
  Old content
</div>
<div class="hidden" data-feature-bit="web-copy-2016" data-options="show-if-enabled:true">
  New content
</div>
